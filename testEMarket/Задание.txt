1. Установить Node js если нету
2. Запустить init.cmd (двойной клик мышкой). он запускает команду npm install которая устанавливает все нужные файлы для работы сервера
3. Запустить  server.cmd - это запустит сервер на котром лежит тестовый сайт. Пока консоль будет открыта - сервер запущен и сайт будет доступен по адресу http://localhost:5000/
4. Нужно получить список телефонов (используя XMLHttpRequest) который лежит в базе данных запущенного сервера и вывести его в index.html используя работыу с DOM посредсвом JavaScript.
   Дизайн на свое усмотрение. Код на получение и отображение писать внутри /js/app.js, а стили писать в /css/main.css.
5. Сайт должен иметь две страницы (Схематическая картинка прилагается):
   - Первая страница - список телефонов полученный с сервера (название телефона + картинка телефона + краткое описание телефона) и кнопка "Добавить в корзину". 
   По клику на эту кнопку телефон должен добавиться в корзину.
   - Вторая страница - корзина. Информация о телефоне таже что и на предыдущей, но есть две кнопки "очистить корзину" и "удалить с корзины". Удалить с корзины, удаляет один телефон.
6. Запросы на сервер делать используя XMLHttpRequest.Не забыть перед отправкой запроса установить тип контента в хидерах в application/json используя setRequestHeader('Content-Type', 'application/json'). Используются методы GET, POST и DELETE. Список адресов для управления базами данных на сервере выглядит следующим образом:
	Получить список телефонов - метод: GET, путь: /get_list, доп параметры: нету
	Получить список телефонов внутри корзины - метод: GET, путь: /get_cart_data, доп параметры: нету
	Добавить телефон в корзину - метод: POST, путь: /add_to_cart, доп параметры: {id: айди телефона}
	Удалить телефон с корзины - метод: DELETE, путь: /delete_item_from_cart, доп параметры: {id: айди телефона}
	Очистить корзину - метод: POST, путь: /reset_cart, доп параметры: нету
